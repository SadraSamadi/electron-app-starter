# Electron App Starter

This is a Starter for Electron App.

## Start

* `git clone https://github.com/SadraSamadi/electron-app-starter`
* `cd electron-app-starter`
* `yarn`

## Development

* `yarn run dev`

## Production

* `yarn run prod`
